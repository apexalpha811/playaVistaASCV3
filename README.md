# Playa Vista ASC — Investor Pitch (V3)

Single-file static site. No build step.

## Deploy

1. Import this repo into Vercel as a static project.
2. Framework preset: **Other**. Build command: empty. Output directory: `./`.
3. Vercel auto-deploys on push to `main`.

## Local preview

Open `index.html` directly in a browser, or:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- `index.html` — entire site (HTML + CSS + JS inline)
- `PlayavistaASC.mp4` — hero video, served from same origin

## Before launch

Fill in the two `[INSTRUMENT]` and `[VALUATION]` placeholders in the Investment tab's ask block.
